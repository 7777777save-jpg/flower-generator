# 大学生语料标签采集器

一个本地运行的小工具，用来围绕“大学生 / 当代大学生 / 大学生活”等关键词，采集近期社媒文本，做基础清洗、分词、词频统计，并筛出 20 到 30 个适合线下贴纸互动的短标签。

## 功能

- 支持微博、B 站自动抓取搜索结果页的可见文本
- 预留小红书、抖音抓取入口，适合在已登录会话下尝试抓取
- 支持把复制来的评论、标题、弹幕、正文直接粘贴进工具补料
- 自动筛选 2 到 4 字、带情绪和身份感的标签词
- 按状态类 / 行为类 / 身份类输出可直接用于 KT 板分区的标签清单

## 使用

```bash
cd young-tag-miner
npm install
npm run install:browser
npm start
```

打开 `http://127.0.0.1:3487`

## 说明

- 平台风控经常变化，自动抓取结果可能受到登录、验证码、反爬策略影响。
- 当前实现里，微博和 B 站更适合直接尝试；小红书和抖音建议配合手动粘贴文本一起用。
- 每次分析结果会自动保存到 `young-tag-miner/outputs/`。
